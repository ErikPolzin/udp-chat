---------------------------------
Asynchronous UDP-based chat app
---------------------------------

Application Setup

---------------------------------
Create virtual python enviroment
---------------------------------

1. Run "make venv" to create virtual enviroment
2. Run "venv/bin/activate" to start the virtual enviroment

---------------------------------
Starting the server
---------------------------------

3. Run "make server" to start running the server

---------------------------------
Starting the client GUI
---------------------------------

In a new terminal/computer:

4. Run "make gui" to start the client GUI 

---------------------------------
Delete enviroment
---------------------------------

5. Run "make" to delete the virtual enviroment
